# INTRODUCCIÓN
Este repositorio contiene mi solución al tech test de I4 Solutions.
Con este proyecto, muestro cómo organizo mi código y
estructuro la aplicación para cumplir los requisitos solicitados.

# DESCRIPCIÓN DEL RETO
Se me proporcionó un dataset almacenado en InfluxDB, con registros de
datos climáticos en Guatemala. En esta aplicación uso esos datos para
visualizar información climática relevante a través de un dashboard.


# CÓMO LEVANTAR EL PROYECTO
1. Instala dependencias con:  
   ```bash
   npm install
   ```
2. Inicia el servidor de desarrollo con:  
   ```bash
   npm run dev
   ```
3. Abre tu navegador en la dirección indicada por la consola (generalmente
   http://localhost:5173) para ver la aplicación.
